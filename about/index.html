    Bienvenue sur Rdyx Blog!

    Ce blog est une combinaison de technologies entièrement gratuites et disponibles en open-source.

    Vous y trouverez principalement un peu de tout ce qui touche à mes centres d’intérêts, à savoir du dev (devops/python/js en majeur partie) et du gaming. :)

    En bas du blog se situe mes liens vers les réseaux sociaux, n’hésitez pas à vous en servir pour me contacter si vous le souhaitez ! ;)

    Technologies utilisées

    • Il est basé sur le principe d’un static-website generator (Jekyll plus particulièrement, codé en Ruby).
    • Hébergé sur Github Pages.
    • Le thème est issue de Jasper 2 (lui même basé sur le thème Casper de Ghost).
    • Le système de recherche est issu de Simple Jekyll Search.
    • Le compteur de visite vient de Hit Counter.
    • Bootstrap 4 pour le css (et un poil de js).
    • 2 fonts principales:
      • JetBrain Mono
      • Ruda
    • Un gros travail de personnalisation (disposition html, modifications css et quelques scripts js).

    En vous souhaitant une bonne lecture, Rdyx ;)
